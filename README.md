# Nusan Quiz Flask App

## how to install ?
- git clone https://github.com/tineetech/nusan_quiz
- cd nusan_quiz
- python -m venv venv
- venv\Scripts\activate
- pip install -r requirements.txt
- python app.py
- open your browser and  try http://127.0.0.1:5000
